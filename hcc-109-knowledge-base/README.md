# HCC 109 Knowledge Base

This repository organizes evidence and working analysis for Halton Condominium Corporation No. 109, 3122 Lakeshore Road West, Oakville, with a focus on Unit 29, records requests, special assessments, reserve fund work, engineering history, and restoration project communications.

## Purpose

The purpose of this knowledge base is to turn scattered email, notice, attachment, and project information into a structured evidence package that can be used for:

- owner meeting preparation;
- records request planning;
- Board and management questions;
- special assessment review;
- restoration project chronology;
- Unit 29 issue tracking;
- exhibit preparation.

## Current Evidence Sources

Primary source material currently comes from Gmail messages labelled `HCC`, including emails and attachments from:

- Maple Ridge Community Management / MRCM;
- John McKenzie;
- Robson Carpenter LLP / Michelle Kelly;
- MRC Customer Service;
- Yardi / MRCM statements;

## Evidence Standard

Each file should separate:

1. confirmed facts;
2. source documents;
3. open questions;
4. possible arguments;
5. records still needed.

This avoids mixing speculation with evidence.

## Important Current Findings

- Unit 29 was identified by the Board as part of the group urgently requiring reserve fund work.
- Units 28-29 were scheduled for the 2026 phase, while the 2025 Avangarde schedule covered Units 23-25 and 15-19 only.
- Synergy inspected garages, terraces, and backyards in August 2024 to assess common element deficiencies.
- The Board stated Synergy helped specify/tender the reserve fund work and Avangarde Restoration Corp. was awarded the contract.
- The records request form submitted in December 2025 appears to contain a date-field defect: the requester date shows `1980/09/26`.
- John McKenzie later allowed the corrected records request to be emailed directly to him.
