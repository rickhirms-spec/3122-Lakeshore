# Workmanship Concerns

## Current Concerns

- Water pooling near threshold.
- Only one visible drainage opening / concern about drainage adequacy.
- Gap at threshold.
- Excess sealant or uneven sealant application.
- Reuse of damaged/chipped pavers or stones.
- Leveling/surface finish concerns.

## Requested Action

- Management/Board inspection before closeout.
- Engineer or qualified reviewer confirmation if issue relates to drainage/waterproofing.
- Written response identifying whether each item is accepted, corrected, deferred, or outside scope.

## Evidence To Add

- Photos with dates.
- Video of pooling after rain if available.
- Notes from contractor/management conversations.
- Final inspection report if received.
