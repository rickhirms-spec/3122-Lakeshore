# Owner-Funded Paver Replacement Request

## Issue

Owner requested permission to replace pavers/stones at owner cost. Current status has not been clearly confirmed in writing.

## Required Status Categories

Management/Board should confirm whether the request is:

- approved;
- denied;
- deferred;
- under Board review;
- outside management authority;
- requiring further information.

## If Denied

Request written reason and criteria relied upon.

## If Approved

Request process, contractor coordination, timing, materials, warranty, and owner cost/responsibility terms.
